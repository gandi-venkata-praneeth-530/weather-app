# 🌦️ Weather App

A simple and responsive weather application that allows users to search for current weather information by city using a clean user interface. Built using HTML, CSS, JavaScript, and integrated with a live weather API.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Displays temperature, humidity, wind speed, and conditions  
- 🖼️ Dynamic weather icons based on the forecast  
- 📱 Responsive design for mobile and desktop  
- 🌐 Real-time data powered by a weather API (e.g., OpenWeatherMap)

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- [OpenWeatherMap API](https://openweathermap.org/api)
